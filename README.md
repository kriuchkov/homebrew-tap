# Homebrew Tap

A collection of Homebrew casks and formulas by [@kriuchkov](https://github.com/kriuchkov).

## Installation

```sh
brew tap kriuchkov/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `linkswitch` | Ultra-lightweight native link router for macOS |

### Install a cask

```sh
brew install --cask linkswitch
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `tock` | Tock is a powerful time tracking tool for the command line. |

### Install a formula

```sh
brew install tock
```

## Updating

```sh
brew update
brew upgrade --cask linkswitch
```
